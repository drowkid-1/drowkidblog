Estos dos scripts son principalmente creados en Shell/bash (lo que los vuelve prácticamente más fáciles de entendimiento), por ello, basta con presionar 2 teclas y creas un usuario ssh capaz de soportar 999 conexiones SIMULTÁNEAS con duración de HASTA 1 AÑO!

# Instalador / Comando de instalación 

Para instalar VPS-LATAM copia y pega el siguiente comando en tu terminal.
```
wget https://raw.githubusercontent.com/ModVPS/LATAM-OFICIAL/main/LATAM; chmod 777 LATAM; ./LATAM --start
```

Para instalar LACASITAMX copia y pega el siguiente comando en tu terminal.

```
apt update -y; apt upgrade -y; wget https://raw.githubusercontent.com/lacasitamx/INSTALADOR-X/master/LACASITA/LACASITA.sh; chmod 777 LACASITA.sh; ./LACASITA.sh 
```
